# Luminary - Kinetic Brutalist Photography Portfolio

A high-performance, avant-garde photography portfolio built with React and Vite. This project features a unique "Kinetic Brutalist" aesthetic with velocity-based animations, massive typography, and physics-driven interactions.

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository (or navigate to the project directory).
2. Install dependencies:

```bash
npm install
```

### Running Locally

Start the development server:

```bash
npm run dev
```

Visit `http://localhost:5173` to view the application.

### Building for Production

Build the optimized application:

```bash
npm run build
```

The output will be in the `dist` directory.
